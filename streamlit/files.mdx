Streamlit provides straightforward widgets to read inputs from files, download results, and preview media.

---

## 1. File Uploads & Image Previews
To preview uploaded images:

```python
import streamlit as st
from PIL import Image

uploaded_img = st.file_uploader("Upload an Image:", type=["jpg", "png"])

if uploaded_img is not None:
    # Open and render image preview
    image = Image.open(uploaded_img)
    st.image(image, caption="Uploaded Preview Image", use_container_width=True)
```

---

## 2. Downloading Files
Allow users to download dataframes, CSV exports, or text results:

```python
import streamlit as st

report_text = "Analysis Results:\n- Accuracy: 94%\n- Status: Stable"

# Provide download button
st.download_button(
    label="Download Report as Text",
    data=report_text,
    file_name="report.txt",
    mime="text/plain"
)
```
